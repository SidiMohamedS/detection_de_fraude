Description du script pour la collecte de données

1. Créer un répertoire et copier le fichier de configuration:

mkdir ~/.kaggle: Cette ligne crée un répertoire nommé .kaggle dans le répertoire personnel de l'utilisateur (~). Le point (.) devant le nom du répertoire indique qu'il est caché.
cp kaggle.json ~/.kaggle/: Cette ligne copie le fichier kaggle.json dans le répertoire nouvellement créé ~/.kaggle. Ce fichier contient probablement votre clé API Kaggle, nécessaire pour accéder aux données.
2. Modifier les permissions du fichier de configuration:

chmod 600 ~/.kaggle: Cette ligne modifie les permissions du fichier ~/.kaggle pour en restreindre l'accès. Le code 600 attribue les permissions suivantes :
Propriétaire (vous) : lecture, écriture
Groupe : aucun accès
Autres : aucun accès
3. Télécharger l'ensemble de données Kaggle:

kaggle datasets download -d mlg-uIB ealaxi/paysim1: Cette ligne utilise la commande kaggle datasets download pour télécharger l'ensemble de données nommé "paysim1" de l'utilisateur "ealaxi" et du groupe "mlg-uIB".
4. Décompresser l'archive (si nécessaire):

unzip paysim1: Cette ligne suppose que l'ensemble de données téléchargé est compressé au format ZIP. Si c'est le cas, cette ligne décompresse le fichier "paysim1.zip" dans le répertoire courant.
