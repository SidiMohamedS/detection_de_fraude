Description du script pour la collecte de données

- Installation du package kaggle, qui est l'API Python officielle de Kaggle
Elle permet aux utilisateurs d'interagir avec la plateforme Kaggle directement à partir de leur environnement Python. 
L'API fournit des fonctionnalités telles que le téléchargement de jeux de données, la gestion des ensembles de données.

Vous pouvez installer l'API Kaggle en utilisant la commande !pip install kaggle dans un environnement compatible avec Python (comme Jupyter notebooks, Google Colab, etc.). Une fois installée, vous devrez configurer l'API 
avec vos informations d'identification Kaggle en utilisant un fichier kaggle.json contenant votre nom d'utilisateur et votre clé d'API.

Création d'un répertoire qui va contenir le fichier à télécharger
Télécharger le fichier kaggle.json depuis votre ordinateur utilisateur

!mkdir -p /root/.kaggle : Cette commande crée le répertoire .kaggle dans le répertoire /root. L'option -p permet également de créer les répertoires parents nécessaires au besoin.

!mv kaggle.json /root/.kaggle/kaggle.json : Cette commande déplace le fichier kaggle.json vers le répertoire /root/.kaggle/. Elle renomme également le fichier, le plaçant dans le bon emplacement pour être utilisé par l'API Kaggle.

Ces commandes sont couramment utilisées lors de la configuration de l'API Kaggle dans un environnement tel que Google Colab.
Une fois que le fichier kaggle.json est correctement configuré dans le répertoire .kaggle, 
vous pouvez utiliser l'API Kaggle pour télécharger des jeux de données en indiquant le nom de l'auteur et du dataset.



- Définir les autorisations du fichier kaggle.json de sorte qu'il soit accessible uniquement par le propriétaire (root) et personne d'autre, avec la commande suivante :
!chmod 600 /root/.kaggle/kaggle.json

les données téléchargées sont sous forme de fichier ZIP. Utilisez la commande !7z e paysim1.zip -odata pour décompresser la données et le charger dans le repertoire data
